Symbee eliminates the need for you to ever send a file to anybody, every upload the file to any server or website, ever email or ftp the file. Just share the files and let your buddies pick up the files they like and when they like. You do the same. Don't ask your buddies to send files, just browse and search the shared folders and pick up the files you like. Browse through the photos shared by your buddies as if they were on your machine, preview each photo and choose what to save and what to skip.

Why do I need to install Java?

Symbee is developed completely in Java. Java is the platform of choice for writing a secure cross-platform application. In addition Java Web Start enables safe and secure way of deploying the application to user's machines. You can find out more about Java at java.com.

How do I allow specific buddies and block others from accessing my shared folders?

Navigate to Settings->Buddy Access.

You can allow specific buddies by entering comma separated list of Google Talk Buddies in the "Allow Buddies: "

You can block buddies by entering a comma separated list of Google Talk Buddies in the "Block Buddies: " section. Please note that "Allow Buddies" takes precedence over "Block Buddies" i.e. If you have "Allow Buddies" set, then Symbee will ignore the "Block Buddies" setting and only allow the buddies listed in "Allow Buddies"

How do I allow only specific file types to be shared?

Navigate to Settings->Sharing Access

You can allow specific file types by entering comma separated list of file extensions in the field "Share files of following types only".

You can certain file types by entering a comma separated list of file extensions in the "Block Buddies: " section. Please note that Allowed file types takes precedence over blocked file types i.e. If you have specified a list of file extensions to be allowed, then Symbee will ignore the blocked file extensions setting and only share the files with extensions allowed.

How do I block users from being able to search my shared folders?

You can turn off searching by navigating to Settings -> Sharing Access.

I have Google Desktop on my machine, however I don't want my buddies to be able to use Google Desktop to search my shared folders?

You can turn off searching using Google desktop by navigating to Settings -> Sharing Access.
