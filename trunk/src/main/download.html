What do I need to run Symbee on my computer?

You will need  Java Runtime 1.6 or higher to be able to run Symbee on your computer. You can download Java Runtime from Sun here.

How do I install Symbee?

Once you have the Java Runtime installed, Click here to install Symbee. The Java runtime environment will download the required files and install the application on your computer.

You may be prompted to select the program to launch the application. Click OK.

You may be prompted to grant access to Symbee as shown in the screen shot below. Please check "Always trust content from this publisher" to avoid being prompted each time you launch Symbee and click Run.
