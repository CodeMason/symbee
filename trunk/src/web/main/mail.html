Subject: Try this app - Share, browse, search - Never email, attach, upload, ftp......

Folks,

We spend a lot of time uploading/downloading photos, music and various other file types and more often than not waiting for your buddy to upload it or send it to you. I have been working on this tool for quite some time now and handpicked a few folks (about 40) to release the first beta out. Its a peer to peer folder sharing tool, but unlike other tools, it is more about sharing the folders with your buddies (Google Talk only as of now). You can browse folders, photos (preview thumbnails), search files that your buddies share at a speed as if they were  on your computer.

I named the tool Symbee - derived from the word Symbiotic i.e. to collaborate - an attribute of a peer-to-peer application.

the basic philosophy is to eliminate the need for you to ever send a file to anybody, ever upload the file to any server or website, ever email or ftp the file. Just share the files and let your buddies pick up the files they like and when they like. You do the same. Don't ask your buddies to send files, just browse and search the shared folders and pick up the files you like. Browse through the photos shared by your buddies as if they were on your machine, preview each photo and choose what to save and what to skip.

Some of the better features of the app are:

  • Share Folders [Screen Shot]
    Share your folders with your Google Talk IM Buddies. Login with your Google Talk account and start sharing folders with you Google Talk buddies. Once you share folders, your buddies will be able to instantly browse through your shared folders.
  • Search [Screen Shot]
    Search the folders that your buddies have shared. If your buddies are using Google desktop, Symbee uses Google desktop to provide the search results.
  • Photo Preview and Sharing [Screen Shot]
    View image previews before you download them. Browse and preview all the photos and choose what to download and what to skip.
  • Download Directories/Multiple Files [Screen Shot]
    You can select multiple files or whole directories to be copied
  • Allow or Block buddies selectively
    You can setup Symbee to allow only specific buddies to access you shared folders or block specific buddies.
  • Allow or block sharing specific file types
    You can setup Symbee to allow only specific files types like music files or photos or block specific file types like word documents.

How do I install and use Symbee

You need Java 1.6 on your machine to run Symbee. Most of the users already have Java running on their machine due to the auto-update features.

  • For Internet Explorer users, just click here, It will install Java if needed and launch symbee. An icon on the Desktop and Program Files will be created.
  • For other browsers, you will have to manually install Java 1.6.
          - You can find a suitable version of Java here
          - After you have installed Java, click here to install symbee
  • For Apple users - I don't know what went wrong, but they yanked the 1.6 version from the Apple's site. However there is still one available here:  http://images.spelpunt.nl/javase6release1dp6.dmg
